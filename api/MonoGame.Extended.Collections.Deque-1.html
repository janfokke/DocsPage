Class Deque<T>

Deque()

Initializes a new instance of the Deque<T> class that is empty and has the default initial capacity.

Declaration
public Deque()
Remarks

The capacity of a Deque<T> is the number of elements that the Deque<T> can hold. As elements are added to a Deque<T>, Capacity is automatically increased by ResizeFunction as required by reallocating the internal array.

If the size of the collection can be estimated, using the Deque(Int32) constructor and specifying the initial capacity eliminates the need to perform a number of resizing operations while adding elements to the Deque<T>.

The capacity can be decreased by calling the TrimExcess() method or by setting the Capacity property explicitly. Decreasing, or increasing, the capacity reallocates memory and copies all the elements in the Deque<T>.

This constructor is an O(1) operation.

TrimExcess()

Sets the capacity to the actual number of elements in the Deque<T>, if that number is less than a threshold value.

Declaration
public void TrimExcess()
Remarks

This method can be used to minimize the Deque<T>'s memory overhead if no new elements will be added. The cost of reallocating and copying the elements of a Deque<T> can be considerable. However, the TrimExcess() method does nothing if the Count is more than 90% of Capacity. This avoids incurring a large reallocation cost for a relatively small gain.

If Count is more than 90% of Capacity, this method is an O(1) operation; O(n) otherwise, where n is Count.

To reset a Deque<T> to its initial state, call the Clear() method before calling the TrimExcess() method. Trimming an empty Deque<T> sets Capacity to the default capacity.

The capacity can also be set using the Capacity property.
