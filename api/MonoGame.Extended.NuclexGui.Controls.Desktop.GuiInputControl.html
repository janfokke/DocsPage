Syntax
public class GuiInputControl : GuiControl, IWritable, IFocusable
Remarks

Through this control, users can be asked to enter an arbitrary string of characters, their name for example. Desktop users can enter text through their normal keyboard where Windows' own key translation is used to support regional settings and custom keyboard layouts.

XBox 360 users will open the virtual keyboard when the input box gets the input focus and can add characters by selecting them from the virtual keyboard's character matrix.

Remarks

This is an unusual property for an input box to have. It is retrieved by the renderer and could be used for several purposes, such as lighting up a control when text is entered to provide better visual tracking or preventing the cursor from blinking whilst the user is typing.
