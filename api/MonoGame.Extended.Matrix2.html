Struct Matrix2

Defines a 3x3 matrix using floating point numbers which can store two dimensional translation, scale and rotation information in a right-handed coordinate system.

Remarks

Matrices use a row vector layout in the XNA / MonoGame Framework but, in general, matrices can be either have a row vector or column vector layout. Row vector matrices view vectors as a row from left to right, while column vector matrices view vectors as a column from top to bottom. For example, the Translation corresponds to the fields M31 and M32.

The fields M13 and M23 always have a value of 

0.0f
, and thus are removed from the Matrix2 to reduce its memory footprint. Same is true for the field M33, except it always has a value of 
1.0f
.
