Interface IThumbLocator

Interface which can be established between a control and its renderer to allow a slider control to locate its thumb

Namespace: MonoGame.Extended.NuclexGui.Controls.Desktop
Assembly: MonoGame.Extended.NuclexGui.dll
Syntax
public interface IThumbLocator
Remarks

A renderer can implement this interface and assign it to a control that it renders so the control can ask the renderer for extended informations regarding the look of its text. If this interface is provided, certain controls will be able to correctly place the caret in user-editable text when they are clicked by the mouse.
