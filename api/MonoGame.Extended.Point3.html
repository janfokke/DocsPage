Remarks

A point is a position in three-dimensional space, the location of which is described in terms of a three-dimensional coordinate system, given by a reference point, called the origin, and three coordinate axes.

A common three-dimensional coordinate system is the Cartesian (or rectangular) coordinate system where the coordinate axes, conventionally denoted the X axis, Y axis and Z axis, are perpindicular to each other.
