ThumbStickDeltaTreshold

The treshold of movement that has to be met in order for the listener to fire an event with the thumbstick's updated position.

In essence this defines the event's resolution.

At a value of 0 this will fire every time the thumbstick's position is not {x:0, y:0}.

Send a vibration command to the controller. Returns true if the operation succeeded.

Motor values that are unset preserve their current vibration strength and duration.

Note: Vibration currently only works on select platforms, like Monogame.Windows.
