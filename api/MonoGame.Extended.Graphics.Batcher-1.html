Parameters
Type Name Description
TDrawCallInfo drawCall

The draw call information.
Remarks

If possible, the drawCall is merged with the last enqueued draw call information instead of being enqueued.

If the enqueue buffer is full, a Flush() is invoked and then afterwards drawCall is enqueued.
